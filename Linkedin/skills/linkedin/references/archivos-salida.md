# Archivos de salida

Crear una carpeta por idea:

`Linkedin/posts/YYYY-MM-DD-nombre-de-la-idea/`

## `IDEA.md`

Estructura:

```md
# Idea

## Titulo interno

## Pilar de contenido

## Objetivo

## Audiencia

## Contexto o disparador

## Idea central

## Enfoque recomendado

## Imagen

## Datos a confirmar
```

## `POST.md`

Estructura:

```md
# Post LinkedIn

## Version principal

## Version alternativa

## CTA o pregunta final

## Hashtags sugeridos

## Notas de edicion
```

## `IMAGEN.md`

Crear solo si aplica.

```md
# Imagen

## Objetivo visual

## Tipo de imagen

## Prompt o instrucciones

## Formato sugerido

## Notas de privacidad
```

## `FUENTES.md`

Crear cuando haya investigacion actual.

```md
# Fuentes

## Fecha de investigacion

## Fuentes consultadas

## Datos usados

## Datos descartados o pendientes de confirmar
```

## `DRIVE.md`

Crear si se prepara o realiza subida a Drive.

```md
# Drive

## Carpeta destino

## Archivos subidos

## Enlaces

## Pendientes
```
