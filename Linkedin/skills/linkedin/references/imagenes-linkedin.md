# Imagenes para LinkedIn

## Regla principal

La imagen debe reforzar la publicacion. Si no suma claridad, confianza o contexto, recomendar publicar solo texto.

## Tipos de imagen

### Foto propia del usuario

Usar cuando el post trate sobre:

- eventos
- reuniones
- networking
- backstage de Aftercode
- aprendizajes personales
- hitos reales

Si el usuario aporta una foto, preguntar o inferir:

- que debe destacarse
- si hay que preservar personas, marcas o datos sensibles
- si se busca un estilo natural o mas profesional

### Imagen generada

Usar cuando el post trate sobre:

- conceptos de IA
- automatizacion
- procesos de negocio
- tecnologia aplicada
- sistemas internos

Estilo recomendado:

- profesional
- moderno
- limpio
- cercano al mundo software/empresa
- sin exceso de futurismo
- sin texto incrustado salvo pedido explicito

Evitar:

- manos irreales
- robots genericos
- imagenes demasiado oscuras
- estilo stock frio
- dashboards con datos legibles inventados
- logos de clientes sin permiso

## Archivo `IMAGEN.md`

Debe incluir:

- objetivo de la imagen
- tipo de imagen recomendada
- prompt usado o propuesto
- formato sugerido
- notas de privacidad o cuidado

Formato sugerido para LinkedIn:

- 1200 x 1200 para imagen cuadrada
- 1200 x 627 para imagen horizontal

Elegir segun el contenido y explicar la recomendacion.
