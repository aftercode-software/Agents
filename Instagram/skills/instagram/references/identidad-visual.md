# Identidad visual Instagram

## Base visual

La identidad de Aftercode en Instagram debe seguir la web:

- fondo negro o casi negro;
- blanco para titulares principales;
- gris claro para bajadas;
- magenta/fucsia fuerte como color de acento;
- bordes finos, lineas y cajas tecnicas;
- composicion moderna, limpia y digital;
- alto contraste;
- tipografia sans bold para titulares y sans regular para textos.

## Sensacion buscada

- desarrollo digital a medida;
- precision;
- tecnologia aplicada;
- productos modernos;
- confianza;
- energia sin parecer infantil.

## Elementos recomendados

- tarjetas oscuras con borde magenta fino;
- iconos lineales simples;
- grillas, lineas, pequenas marcas cuadradas;
- mockups abstractos de sistemas sin datos legibles;
- patrones sutiles de interfaz;
- screenshots o capturas propias solo si no revelan datos sensibles;
- contraste entre problema y solucion.

## Evitar

- robots genericos;
- cerebros luminosos;
- manos deformes o personas stock demasiado pulidas;
- imagenes futuristas sin relacion con negocio;
- dashboards con numeros inventados legibles;
- degradados multicolor fuera de marca;
- exceso de texto en la imagen;
- logos de clientes sin permiso.

## Formatos

- Historias: 1080 x 1920.
- Post cuadrado: 1080 x 1080.
- Carrusel: 1080 x 1350 recomendado para mayor presencia en feed.
- Reel cover: 1080 x 1920 con zona segura centrada.

## Prompt base para imagen generada

Usar como base y adaptar:

```text
Visual para Instagram de una empresa de desarrollo de software a medida llamada Aftercode. Estetica dark premium, fondo negro profundo, acentos magenta/fucsia, lineas finas, tarjetas de interfaz abstractas, sensacion de producto digital moderno y automatizacion empresarial. Sin texto legible dentro de la imagen, sin logos de terceros, sin personas stock, sin robots genericos. Composicion limpia, alto contraste, profesional, B2B tecnologia.
```

Si la pieza necesita texto, proponerlo por separado para maquetar, no incrustarlo en la imagen generada salvo pedido explicito.
